Upload gallery photographs to GitHub.

This page helps an authorised admin upload a photo and description to the GitHub repository. After GitHub Pages republishes, the item appears on the Gallery page.

Before uploading

Use a limited GitHub token.

The website is static, so it cannot safely store a password. Paste a fine-grained GitHub token only when uploading; it stays in this browser session and is never written into the site files.

  • Create a fine-grained token for this repository only.
  • Give it Contents: Read and write permission.
  • Keep the expiry short and revoke it if it is no longer needed.
